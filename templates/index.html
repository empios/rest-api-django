API DOCS

Github Repo

Endpoints:

movies/ -> fetch all movies in db with ratings related to it
add/[film title] -> add a movie with this title from external movie db and insert to ours
comments/ -> fetch all comments
addcomment -> add comments to database with json (POST and with request body in json format) like this {"comment": "Bardzo Bardzo dobry komentarz","movie_id": "3"}
comment/[id of movie] -> fetch comments related to movie id

Installation

  1. Clone this repo
  2. Pip install in terminal
  3. python manage.py runserver
